Prime Number Check

Problem Statement

Determine if a given number is prime.

Input:

An integer n (1 ≤ n ≤ 106)

Output:

"Prime" if n is prime, otherwise "Not Prime".

Algorithm

  1. If n ≤ 1, return "Not Prime"
  2. For i from 2 to √n:
    • If n % i == 0, return "Not Prime"
  3. If no divisors found, return "Prime"

Examples

Input Output Explanation
13 "Prime" 13 has no divisors other than 1 and itself
15 "Not Prime" 15 is divisible by 3 and 5
1 "Not Prime" By definition, 1 is not a prime number
