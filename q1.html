Introduction

A Singly Linked List is a dynamic data structure that consists of a sequence of nodes. Each node contains data and a pointer that stores the address of the next node in the list. It allows efficient insertion and deletion operations.

A linked list does not require contiguous memory like an array. Nodes are created in memory as needed using dynamic memory allocation.

Node Structure

Each node has two parts:

  1. Data field – stores the information.
  2. Next pointer – stores the address of the next node.

Diagram of a Node

+-----------+------------+
|   Data    |   Next     | ---> Points to next node
+-----------+------------+

Representation

The first node of the list is pointed to by a pointer called HEAD. If the list is empty, HEAD = NULL.

Example Linked List

 HEAD
  |
  v
+----+------+     +----+------+     +----+------+     +--------+
| 10 |  o-------->| 20 |  o-------->| 30 |  o-------->|  NULL  |
+----+------+     +----+------+     +----+------+     +--------+

Here:

  • 10, 20, 30 are data
  • Arrows represent pointers storing the address of the next node

Node Definition

In C

struct node {
    int data;
    struct node *next;
};
struct node *head = NULL;

In Java

class Node {
    int data;
    Node next;
}
Node head = null;

How Nodes are Linked

When a new node is created, its memory address is stored in the next pointer of the previous node. This forms a chain-like structure.

Memory allocation is done using:

  • malloc() in C
  • new operator in Java

Advantages

  1. Dynamic Size: The list grows and shrinks based on memory availability.
  2. Efficient Insertions/Deletions: No shifting of elements like arrays.
  3. Memory Utilization: Uses memory effectively.

Disadvantages

  1. No Direct Access: Must traverse to reach an element.
  2. Extra Memory: Pointer takes additional space.
  3. Slower Traversal: Compared to index-based access in arrays.

Applications

  • Implementation of Stacks and Queues
  • Polynomial expressions
  • Adjacency representation in Graphs
  • Memory management systems

Conclusion

A Singly Linked List is a flexible and efficient data structure used where frequent insertion and deletion operations are required. Understanding node structure and pointer manipulation is essential for implementing linked lists effectively.
