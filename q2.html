Palindrome Number Check

Problem Statement

Check if a number reads the same backward as forward.

Input:

An integer n (0 ≤ n ≤ 106)

Output:

"Palindrome" if n is a palindrome, otherwise "Not Palindrome".

Algorithm

  1. Store original = n
  2. Initialize reversed = 0
  3. While n > 0:
    • Extract last digit: digit = n % 10
    • Update reversed number: reversed = reversed * 10 + digit
    • Remove last digit: n /= 10
  4. If original == reversed, return "Palindrome", else "Not Palindrome"

Examples

Input Output Explanation
121 "Palindrome" 121 reads the same backward
123 "Not Palindrome" 123 reversed is 321
1221 "Palindrome" Symmetrical number
