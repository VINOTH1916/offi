Leap Year Check

Problem Statement

Determine if a given year is a leap year.

Input:

An integer year (1 ≤ year ≤ 9999)

Output:

"Leap Year" if it is a leap year, otherwise "Not Leap Year".

Algorithm

  1. If year is divisible by 400:
    • Return "Leap Year"
  2. Else if year is divisible by 100:
    • Return "Not Leap Year"
  3. Else if year is divisible by 4:
    • Return "Leap Year"
  4. Else:
    • Return "Not Leap Year"

Examples

Input Output Explanation
2000 "Leap Year" Divisible by 400
1900 "Not Leap Year" Divisible by 100 but not 400
2024 "Leap Year" Divisible by 4 but not 100
2023 "Not Leap Year" Not divisible by 4
